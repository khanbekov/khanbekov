- Built a backtesting framework compatible with various chart types and multiple signal-processing layers.
- Integrated the backtester with live trading systems, providing seamless deployment of optimized strategies.

- Performed comprehensive performance tuning of trading strategies, leveraging historical data to enhance predictive accuracy and execution efficiency.

### Bots for Automated Trading
- Designed connectors for integrating bots with multiple cryptocurrency exchanges, implementing diverse strategies such as arbitrage, trend-following, and liquidity provision.
- Developed **trader assistant tools** to automate routine tasks and enhance manual trading precision.

### TradingView Indicators and Strategies
- Authored multiple indicators and automated trading strategies for TradingView using Pinescript, tailored to both retail and professional traders.
